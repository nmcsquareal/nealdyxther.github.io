ExoStellar

This is an Astronomy website that mainly uses HTML, CSS and Bootstrap as its front end technology. It also utilizes different API such as Open APIs from NASA API Portal.

The main idea of ExoStellar is for it to be used as an additional educational resource for studying Astronomy. This is going to be helpful for students or any people interested in Science and Astronomy.

Link: ExoStellar

StartUp

StartUp is an educational resource website for studying the basic of front end web development, particularly the HyperText Markup Protocol (HTML). The content was divided into seven (7) topics - Getting Started, Texts, Lists, Images, Links, Table and Forms.

Link: StartUp

Shappy

This is a two way e-commerce website where the admin can sell something and users can buy it. It has functionalities such as a Log-in and Sign-Up page, Products Section, Add to Cart, Order Section and Purchase.

It was named Shappy to emphasize that this is an e-commerce website that mainly focuses on customers' satisfaction and happiness.

Link: Shappy
